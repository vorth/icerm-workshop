Viewing VRML / X3D Models

X3D is a successor to VRML, a standard 3D file format designed in the 1990s. The standard is managed by the Web3D Consortium. This page shows a web component from X_ITE that can display 3D models in VRML or X3D format.

You can right-click to access full-screen and other features in a popup menu.
